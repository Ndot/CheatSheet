HTML Document Division Element

The HTML <div> Element is the generic container for flow content, which does not inherently represent anything. It should be used only when no other semantic element (such as <article> or <nav>) is appropriate.

Global Attributes

Example
<div style="border: 2px solid; padding: 15px;">
 <p>I am inside a div tag.</p>
 <div style="border: 2px solid; padding: 15px;">
  <p>And so am I.</p>
 </div>
</div>

I am inside a div tag.
