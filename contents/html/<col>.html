HTML Table Column Element

The HTML <col> Element defines a column within a table. Permitted only inside a <colgroup> tag it is useful for applying styles to entire columns.

Attributes :

span

Example
<table>
 <colgroup>
  <col span="2" style="background-color: hsl(209, 56%, 45%)">
  <col span="1">
  <col span="1" style="background-color: hsl(0, 0%, 55%)">
 </colgroup>
 <tr>
  <th>Lime</th>
  <th>Lemon</th>
  <th>Orange</th>
  <th>Red</th>
 </tr>
 <tr>
  <td>Green</td>
  <td>Yellow</td>
  <td>Orange</td>
  <td>Red</td>
 </tr>
</table>
Result
Lime Lemon Orange Red
Green Yellow Orange Red
