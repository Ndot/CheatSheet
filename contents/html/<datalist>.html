HTML Datalist Element

The HTML <datalist> Element contains a set of <option> tags that represent the values available for other controls.

Attributes :

Global Attributes

Example
<label>Choose a color:</label>
<input list="colors" name="myBrowser" placeholder="Colors"/>
<datalist id="colors">
	<option value="Red">
	<option value="Blue">
	<option value="Orange">
	<option value="Green">
	<option value="Yellow">
</datalist>
