HTML Abbreviation Element

Represents an abbreviation with a full description on the title attribute.

Attributes :

title

Example
<p>Hover over HTML to see the result.</p>
<p>I do <abbr title="Hypertext Markup Language">HTML.</abbr></p>
Result

Hover over HTML to see the result.


I do HTML.
