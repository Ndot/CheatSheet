HTML Audio Element

The <audio> element is used to embed sound content in your html document. Fallback content for non supporting browsers can be added inside the audio block.

The supported file formats are MP3, Wav and Ogg.

Use of src attribute in the <audio> element is optional you can use the <source> element inside the audio block to specify the audio to embed.

Attributes :

autoplay, buffered, controls, loop, muted, played,preload, src, volume

Example
<audio controls="controls">
 Your browser does not support this audio element.
 <source src="./audio/1.ogg" />
</audio>
