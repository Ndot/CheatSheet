HTML Deleted Text Element

The HTML <del> Element represents a range of text that has been deleted from a document.

Attributes :

cite, datetime

Example
<p><del datetime="1996-04-24" >This is way to old.</del> So it had to go.</p>
Result

This is way to old. So it had to go.
