HTML Meta Element

The HTML <meta> Element represents any metadata information that cannot be represented by one of the other HTML meta-related elements like <base>, <link>, <script>, <style> or <title>.

Note that the global name has a specific semantic in the <meta> element and that the itemprop must not be set when one of the name, http-equiv or charset is already used.

The start tag must be present and the end tag must NOT be present.

Attributes :

charset, content, http-equiv, name

Example
<meta charset="utf-8">

<!-- Please always use this -->
<meta name="viewport" content="width=device-width">
