HTML Ordered List Element

The HTML <ol> Element represents an ordered list of items, typically rendered as a numbered list.

Attributes :

reversed, start, type

Example
<ol start="7">
    <li>first item</li>
    <li>second item</li>
    <li>third item</li>
</ol>
Result
  1. first item
  2. second item
  3. third item
