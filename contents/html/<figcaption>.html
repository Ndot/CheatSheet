HTML Figure Caption Element

HTML <figcaption> Element defines a caption for a <figure> element. This element is optional.

Attributes :

Global Attributes

Example
<figure>
 <img src="./icons/logo_page.png" alt="Logo" />
 <figcaption>This is a logo.</figcaption>
</figure>
