HTML <!DOCTYPE> Declaration

The <!DOCTYPE> declaration is not an HTML tag; it is an instruction to the web browser about what version of HTML the page is written in. This is the first thing on your HTML document.

With HTML 5 this became very symplified. The example below shows how simple it is to write an HTML document.

Attributes :

null

Example
<!DOCTYPE html>
<html lang="en">
 <head>
  <meta charset="UTF-8">
  <title>Title</title>
 </head>
 <body>
  <h1>Thats all folks!!!</h1>
 </body>
</html>
