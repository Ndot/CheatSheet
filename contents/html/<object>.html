HTML Object Element

The HTML <object> Element represents an embedded resource within an HTML document, which can be treated as an image, a nested browsing context, or a resource to be handled by a plugin.

Attributes :

data, height, name, type, usemap, width

Example
<object data="/404.html" type="text/html" width="100%"></object>
