HTML Canvas Element

The HTML <canvas> Element can be used to draw graphics via scripting. For example, it can be used to draw graphs, make photo compositions or even perform animations. You should provide alternate content inside the <canvas> block. That content will be rendered in older browsers or browsers with JavaScript disabled.

Attributes :

height, width

Example
<canvas id="myCanvas">
Your browser does not support the canvas tag.
</canvas>
