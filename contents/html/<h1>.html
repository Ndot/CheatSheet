HTML Heading Elements

Heading elements implement six levels of document headings, <h1> to <h6> tags are used to define HTML headings. The <h1> defines the most important heading, <h6> defines the least important heading.

Attributes :

Global Attributes

Example
<h1>Heading level 1</h1>
<h2>Heading level 2</h2>
<h3>Heading level 3</h3>
<h4>Heading level 4</h4>
<h5>Heading level 5</h5>
<h6>Heading level 6</h6>
