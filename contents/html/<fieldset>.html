HTML Fieldset Element

The HTML <fieldset> Element is used to group several controls as well as <label> tags within a web form.

Attributes :

disable, form, name

Example
<form action="#" method="POST">
	<fieldset>
		<legend>Title</legend>
		<label>Name </label><input type="text">
		<fieldset>
			<legend>Title in Title</legend>
			<label>Name </label><input type="text">
		</fieldset>
	</fieldset>
</form>
Result
Title
Title in Title
