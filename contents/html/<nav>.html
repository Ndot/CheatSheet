HTML Nav Element

The HTML <nav> Element represents a section of a page that links to other pages or to parts within the page.

This element is intended only for major blocks of navigation links.

A document can have several <nav> elements, for example, one for site navigation and one for intra-page navigation.

Attributes :

Global Attributes

Example
<nav>
	<a href="#html">HTML</a>
	<a href="#css">CSS</a>
	<a href="#javascript">JavaScript</a>
</nav>
