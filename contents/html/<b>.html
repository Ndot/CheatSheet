HTML <b> Element

The HTML <b> Element represents a span of text stylistically different from normal text.

This element should be of last resort. You should prefer <h1> to <h6> for titles, <em> for emphasized, <strong> for important and <mark> for marked or highlighted text. Text inside this element results in bold text.

Attributes :

Global Attributes

Example
<p>Normal text. <b>Text inside <b> element.</b></p>
Result

Normal text. Text inside <b> element.
