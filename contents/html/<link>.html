HTML Link Element

The HTML <link> Element specifies relationships between the current document and an external resource. Possible uses for this element include defining a relational framework for navigation. This Element is most used to link to style sheets.

The rel attribute can be set to a lot of different values. They are listed in this separate page.

Attributes :

crossorigin, disable, href, hreflang, media, methods, rel, sizes, target, title, type

Example
<!-- This stylesheet is PERSISTENT -->
<link href="index.css" rel="stylesheet">

<!-- This stylesheet is PREFERRED -->
<link href="default.css" rel="stylesheet" title="Default Style">

<!-- This stylesheets are ALTERNATIVE -->
<link href="fancy.css" rel="alternate stylesheet" title="Fancy">
<link href="basic.css" rel="alternate stylesheet" title="Basic">
