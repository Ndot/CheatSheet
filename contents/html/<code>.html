HTML Code Element

The HTML <code> Element represents a fragment of computer code. By default, it is displayed in the browser's default monospace font.

Attributes :

Global Attributes

Example
<code>
 <!-- Do some code here... -->
 if (the code works) {
  do this
 } else {
  do that
 }
</code>
Result
if (the code works) { do this } else { do that }
