HTML Area element

The HTML <area> element defines a hot-spot region(clickable area) on an image, and optionally associates it with a hypertext link. This element is always nested within a <map> element. The Area element is a Self-Closing Tag.

Attributes :

alt, coords, download, href, hreflang, media, rel, shape, target, type

Example
<img src="/assets/media/sample_area_tag.png" alt="Image Sample" height="200px" usemap="#primary">
<map name="primary">
	<area shape="circle" coords="100,100,35" href="#html/home" title="Let's go Home"/> 
</map>


<p>The blue circle on the Icon will take you to the HTML Home on this page</p>
