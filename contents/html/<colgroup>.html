HTML Table Column Group Element

The HTML <colgroup> Element defines a group of columns within a table for formatting.

Attributes :

span

Example
<table>
 <colgroup span="1" style="background-color: hsl(0, 0%, 55%)"></colgroup>
 <tr>
  <th>Lime</th>
  <th>Lemon</th>
 </tr>
 <tr>
  <td>Green</td>
  <td>Yellow</td>
 </tr>
</table>
Result
Lime Lemon
Green Yellow
