HTML Base Element

The <base> tag specifies the base URL/target for all relative URLs in a document.

Normally the <base> element shows as the first element inside the <head> element this way all other elements on the page will use it's information.

The base URL of a document can be queried from a script using document.baseURI.

Attributes :

href, target

Example
<base href="http://www.example.com/someFolder/" target="_blank">
Result

The code above will do two things.

Make all relative paths look inside someFolder
Make all links open as if they had an attribute of target=_blank
