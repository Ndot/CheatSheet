HTML Description List Element

The HTML <dl> Element encloses a list of pairs of terms and descriptions. Common uses for this element are to implement a glossary or to display metadata (a list of key-value pairs).

Attributes :

Global Attributes

Example
<!-- Single term and description -->
<dl>
	<dt>Firefox</dt>
	<dd>A free, open source, cross-platform, graphical web browser.</dd>
</dl>
<hr />
<!-- Multiple terms, single description -->
<dl>
	<dt>Oranges</dt>
	<dt>Apples</dt>
	<dd>All fruits.</dd>
</dl>
<hr />
<!-- Single term, multiple descriptions -->
<dl>
	<dt>Orange</dt>
	<dd>A fruit of the citrus species, round, orange and delicious.</dd>
	<dd>A color.</dd>
</dl>
