HTML Anchor Element

Defines an hyperlink to a location on the same page or any other page on the Web. It can also be used to create an anchor point to some place within the contents of the same page.

Attributes :

download, href, hreflang, ping, rel, target, type

Example
<a href="https://developer.mozilla.org/en-US/" target="_blank">MDN Mozilla Developer.</a>

<a href="#html">This will take you to the HTML Home on this page.</a>
Result
MDN Mozilla Developer.

This will take you to the HTML Home on this page.
