HTML <body> Element

The HTML <body> Element represents the content of an HTML document. There can be only one <body> element in a document.

Attributes :

onafterprint, onbeforeprint, onbeforeunload, onblur, onerror, onfocus, onhashchange, onload, onmessage, onoffline, ononline, onpopstate, onredo, onresize, onstorage, onundo, onunload

Example
<!DOCTYPE html>
<html lang="en">
	<head>
		<meta charset="UTF-8">
		<title>Title</title>
	</head>
	<body>
		<h1>Body</h1>
		<p>This is where your page content will go.</p>
	</body>
</html>
