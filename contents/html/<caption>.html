HTML Caption Element

The HTML <caption> Element represents the title of a table. It is always the first element of a table tag, but using CSS it can be place elsewhere, relative to the table.

Attributes :

Global Attributes

Example
<table>
	<caption>This is a Caption</caption>
	<tr>
		<th>Header 1</th>
		<th>Header 2</th>
	</tr>
	<tr>
		<td>Data 1</td>
		<td>Data 2</td>
	</tr>
</table>
Result
This is a Caption
Header 1 Header 2
Data 1 Data 2
