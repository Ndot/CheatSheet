HTML Label Element

The HTML Label Element <label> represents a caption for an item in a user interface. If the user clicks on the text within the <label> element, it toggles the control. The for attribute of the <label> tag should be equal to the id attribute of the related element to bind them together.

Attributes :

for

Example
<label>Name: <input type="text" placeholder="Your name here..."></label>

<label for="name">Name: </label>
<input type="text" id="name" placeholder="Or maybe here...">
