HTML Bi-Directional Isolation Element

The HTML <bdi> Element isolates a span of text that might be formatted in a different direction from other text outside it.

Note the dir attribute is not inherited. If not set, its default value is the auto which let the browser decide the direction based on the element's content.

Attributes :

Global Attributes

Example
<bdo dir="rtl">This text <bdi>SOME TEXT HERE</bdi> is automatically displayed left-to-right.</bdo>

<p>The text inside the bdi tag is left-to-right. Because that is the default.</p>
<p>Everything else follows the bdo direction attribute.</p>
Result
This text SOME TEXT HERE is automatically displayed left-to-right.
