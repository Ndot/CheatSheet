HTML Description Element

The HTML <dd> Element indicates the description of a term in a <dt> tag (Definition Term Element).

Attributes :

Global Attributes

Example
<dl>
	<dt>Oranges</dt>
	<dd>Are fruits of the citrus species, they are round, orange and delicious.</dd>
	<dt>Apples</dt>
	<dd>They are ok.</dd>
</dl>
Result
Oranges
Are fruits of the citrus species, they are round, orange and delicious.
Apples
They are ok.
