HTML Details Element

The HTML <details> Element specifies additional that can be used to create an interactive widget that the user can open and close.

The attribute open="open" defaults to false, so if ommited the <details> content will be hidden.

If the <details> tag is supported, the <summary> tag gives a title to the <details> tag, if ommited you should see instead the word Details.

You should not see the content within the <details> tag. If you do your browser either does not support the tag or as it disabled.

Attributes :

open

Example
<details>
 <summary>Some details about something...</summary>
 <p>More info about the details.</p>
 <p>Even more info about the details.</p>
 <p>And some more info about the details.</p>
</details>
Result

More info about the details.

Even more info about the details.

And some more info about the details.
