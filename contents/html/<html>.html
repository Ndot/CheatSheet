HTML Root Element

The HTML <html> Element represents the root of an HTML document. All other elements must be descendants of this element. Although this tag can be implied, or not required, with HTML, it is required to be opened and closed in XHTML.

Attributes :

Global Attributes

Example
<!DOCTYPE html>
<html>
	<head>...</head>
	<body>...</body>
</html>
