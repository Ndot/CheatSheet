HTML Definition Element

The HTML <dfn> Element represents the defining instance of a term.

Attributes :

Global Attributes

Example
<dl>
 <dt>
  <dfn id="html-definition"><abbr title="HyperText Markup Language">HTML</abbr></dfn>
 </dt>
 <dd>Is the standard markup language for creating web pages.</dd>
</dl>
<br />
<hr />
<p>Later in the page you can reference that definition.</p>
<p>Learn some <a href="#html-definition">HTML</a> tomorrow.</p>

HTML
