HTML Header Element

The HTML <header> Element is a container for some introductory elements of a given content. It can contain elements like headings, logos, links, forms, authorship information, etc.

Attributes :

Global Attributes

Example
<header>
	<h1>My Page Title</h1>
	<h3>Some other less important title</h3>
	<p>This is <cite style="color: orange;">Cheat Sheat</cite> page.</p>
</header>
<p>And here I will tell you all about my page...</p>
Result

My Page Title

Some other less important title

This is Cheat Sheat page.

And here I will tell you all about my page...
