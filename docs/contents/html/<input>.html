HTML Input Element

The <input> Element specifies an input field where the user can enter data. An input field can vary in many ways, depending on the type attribute.

Attributes :

type, accept, autocomplete, autofocus, capture, checked, disabled, form, formaction, formmethod, formnovalidate, formtarget, height, inputmode, list, max, maxlength, min, minlength, multiple, name, pattern, placeholder, readonly, required, selectionDirection, size, spellcheck, src, step, tabindex, value, width

Example
<form>
	<label>Text: <input type="text" placeholder="Type something..."/></label><br/>
	<label>Password: <input type="password" /></label><br/>
	<label>Number: <input type="number" value="54" step="34"/></label><br/>
	<label>Button: <input type="button" value="Click Me"/></label><br/>
	<label>File: <input type="file"/></label><br/>
	<label>Color: <input type="color" value="#7CDA33"/></label><br/>
	<label>Reset: <input type="reset" value="Reset Form"/></label><br/>
</form>
