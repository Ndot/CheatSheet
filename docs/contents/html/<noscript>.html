HTML Noscript Element

The <noscript> tag defines an alternate content for users that have disabled scripts in their browser or have a browser that doesn't support script.

Attributes :

Global Attributes

Example
<div>You will be able to see this content.</div>
<noscript>If SCRIPTING is off you will also be able to see this content.</noscript>
Result
You will be able to see this content.
