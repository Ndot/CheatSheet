HTML Definition Term Element

The HTML <dt> Element identifies a term in a definition list.

Attributes :

Global Attributes

Example
<dl>
	<dt>dl</dt>
	<dt>dt</dt>
	<dt>dd</dt>
	<dd>They are a pain in my ***</dd>
</dl>
Result
dl
dt
dd
They are a pain in my ***
