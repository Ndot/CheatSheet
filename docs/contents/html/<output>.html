HTML Output Element

The HTML <output> Element represents the result of a calculation or user action.

Attributes :

for, form, name

Example
<form oninput="result.value = number.value">
    <input id="number" type="range" min="20" max="80" value="25">
    <output name="result">25</output>
</form>
Result
25
