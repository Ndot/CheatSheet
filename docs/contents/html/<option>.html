HTML Option Element

The HTML <option> element is used to create a control representing an item within a <select>, an <optgroup> or a <datalist> HTML5 element.

Attributes :

disabled, label, selected, value

Example
<select>
    <option label="item" value="item01">First</option>
    <option label="item" value="item02">Second</option>
    <option label="item" value="item03" selected>Tird</option>
    <option label="item" value="item04" disabled>Im the last one</option>
</select>
