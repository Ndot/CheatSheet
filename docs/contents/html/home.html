HTML

HTML, which stands for HyperText Markup Language, is the most basic building block of a webpage and used for creating and visually representing a webpage. It determines the content of a webpage, but not its functionality.

Organisation

On your left you can find all the HTML 5 tags organised alphabetically. If no links are visible tap or hover on the top left button.

You can also look up the tags grouped by their function by clicking on the Element Reference link.

Most of the information presented on this page comes from the MDN - Mozilla Developer Network and at the end of each element content you will find a link to the full documentation on their site.
