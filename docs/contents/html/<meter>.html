HTML Meter Element

The <meter> Element defines a scalar measurement within a known range, or a fractional value.

The <meter> tag should not be used to indicate progress (as in a progress bar). For progress bars, use the <progress> tag.

Attributes :

form, high, low, max, min, optimum, value

Example
<p>My value is 10 <meter min="0" max="100" low="20" high="75" optimum="100" value="10"></meter></p>
<p>My value is 40 <meter min="0" max="100" low="20" high="75" optimum="100" value="40"></meter></p>
<p>My value is 60 <meter min="0" max="100" low="20" high="75" optimum="100" value="60"></meter></p>
<p>My value is 90 <meter min="0" max="100" low="20" high="75" optimum="100" value="90"></meter></p>
Result

My value is 10

My value is 40

My value is 60

My value is 90
